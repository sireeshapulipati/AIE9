## 2-Week Exercise Program for Alex

### Overview
- **Duration:** 2 weeks
- **Frequency:** 3 workouts per week (Monday, Wednesday, Friday)
- **Duration per Workout:** 30 minutes
- **Focus:** Building consistency, improving energy levels, incorporating calming elements for anxiety.

---

### Weekly Schedule
**Evening Workouts Recommended**: Choose a time that works best for you, preferably post-dinner. 

---

### Week 1
| Day        | Workout Details                                                                                 |
|------------|------------------------------------------------------------------------------------------------|
| Monday     | **Warm-Up (5 min)**: Gentle dynamic stretches (arm circles, hip circles, leg swings)  
|            | **Workout (20 min)**:  
|            | - Bodyweight Squats: 3 sets of 10-12 reps  
|            | - Seated Dumbbell Shoulder Press: 3 sets of 8-10 reps  
|            | - Plank (on knees if needed): 3 sets of 20-30 seconds  
|            | - Cat-Cow Stretch: 2 mins  
|            | **Cool Down (5 min)**: Deep breathing or seated forward fold.  
| Wednesday  | **Warm-Up (5 min)**: Gentle dynamic stretches  
|            | **Workout (20 min)**:  
|            | - Step-Ups (using a sturdy platform): 3 sets of 10-12 reps each leg  
|            | - Bent Over Dumbbell Rows: 3 sets of 8-10 reps  
|            | - Glute Bridges: 3 sets of 10-12 reps  
|            | - Child’s Pose: 2 mins  
|            | **Cool Down (5 min)**: Deep breathing or gentle stretches focusing on legs and back.  
| Friday     | **Warm-Up (5 min)**: Gentle dynamic stretches  
|            | **Workout (20 min)**:  
|            | - Lunges (stationary or walking): 3 sets of 10 reps each leg  
|            | - Wall Push-Ups: 3 sets of 8-10 reps  
|            | - Bicycle Crunches: 3 sets of 10-12 reps  
|            | - Seated Forward Fold: 2 mins  
|            | **Cool Down (5 min)**: Deep breathing exercises or meditation.  

---

### Week 2
| Day        | Workout Details                                                                                 |
|------------|------------------------------------------------------------------------------------------------|
| Monday     | **Warm-Up (5 min)**: Gentle dynamic stretches  
|            | **Workout (20 min)**:  
|            | - Bodyweight Squats: 3 sets of 12-15 reps (progress)  
|            | - Seated Dumbbell Shoulder Press: 3 sets of 10-12 reps (progress)  
|            | - Side Plank (on knees if needed): 2 sets of 15-20 seconds each side  
|            | - Cat-Cow Stretch: 2 min 
|            | **Cool Down (5 min)**: Focus on deep breathing or seated forward fold.
| Wednesday  | **Warm-Up (5 min)**: Gentle dynamic stretches  
|            | **Workout (20 min)**:  
|            | - Step-Ups: 3 sets of 12-15 reps (progress)  
|            | - Bent Over Dumbbell Rows: 3 sets of 10-12 reps (progress)  
|            | - Glute Bridges: 3 sets of 12-15 reps (progress)  
|            | - Child’s Pose: 2 mins  
|            | **Cool Down (5 min)**: Deep breathing or gentle stretches.
| Friday     | **Warm-Up (5 min)**: Gentle dynamic stretches  
|            | **Workout (20 min)**:  
|            | - Lunges: 3 sets of 12 reps each leg (progress)  
|            | - Incline Wall Push-Ups: 3 sets of 10-12 reps  
|            | - Bicycle Crunches: 3 sets of 12-15 reps  
|            | - Seated Forward Fold: 2 mins  
|            | **Cool Down (5 min)**: Deep breathing exercises or meditation.  

---

### Progression Plan
- **Week 1**: Focus on form and comfort with exercises.  
- **Week 2**: Increase reps within the suggested range as comfort grows, ensuring no discomfort or pain.  
- Include 1-2 minutes of mindful breathing at the end of each session, focusing on relaxing the body and calming the mind.

### Notes
- Ensure proper hydration before, during, and after workouts.  
- If experiencing anxiety during exercises, take slow, deep breaths or pause as needed.  
- Modify any exercises as needed based on comfort level and listen to your body.
